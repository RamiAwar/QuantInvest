Basic Optimizer
What is this?
This is a simplified version of the custom optimizer. This interface helps new investors explore the relationship between expected returns and volatility of portfolios. When a time range, expected returns percentage, and volatility percentage are chosen, the optimizer attempts to find a portfolio that has these statistics. The stocks considered are all current S&P500 stocks, and thus the generated portfolio will contain a subset of those.
How does it work?
Behind the scenes, the optimizer first attempts to find a portfolio that matches the expected returns requested, while minimizing volatility. Then the optimizer attempts to find a portfolio that matches the requested volatility while maximizing returns. The portfolio with statistics closer to those requested by the user is returned.
